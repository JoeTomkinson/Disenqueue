# Resources Mapping

This document is a mapping of .tga image resources by filename to their corresponding location in the addon.

| Filename            | Location in Addon                                                     | Colour Tints                  |
| ------------------- | --------------------------------------------------------------------- | ----------------------------- |
| `bookmark.tga`      | tbd                                                                   | None                          |
| `close-square.tga`  | Close button icon in header of all UI panels                          | (Soft Red) #E56666            |
| `closed-eyes.tga`   | Remove button icon per item row in main UI                            | (Dusty Rose) #D97373          |
| `heart.tga`         | Manually locked items icon in row of Locked Items UI                  | (Red) #FF4D4D                 |
| `import-export.tga` | Import confirm button icon in Import dialog (after text is pasted)    | (Bright Green) #80FF80        |
| `lightning.tga`     | Auto-locked items icon in row of Locked Items UI                      | (Muted Orange) #E69933        |
| `lock-closed.tga`   | Locked Items button in header of main UI & lock button per item row   | (Gold) #FFD14D                |
| `settings-cog.tga`  | Settings button in header of main UI (opens options)                  | (Metal Grey) #B3B8BF          |
| `export.tga`        | Export button in Locked Items UI header & Copy to Clipboard button    | (Teal) #66CCE6                |
| `import.tga`        | Import button in Locked Items UI header & Paste from Clipboard button | (Soft Green) #66D98C          |
